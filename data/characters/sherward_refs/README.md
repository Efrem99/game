# Shervard Reference Pack

Put likeness reference images here for `models/refine_sherward_likeness.py`.

Recommended filenames:
- `front.png`
- `side.png`
- `threeq.png`

Supported image extensions:
- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

You can also pass explicit paths via script arguments:
- `--front-ref`
- `--side-ref`
- `--threeq-ref`
